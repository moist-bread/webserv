More about this page:

Here you can test out the redirection settings in the config.
By clicking on any of those buttons they won't take you to a page with the same name but instead to a different Location.

To do that we use the Location response header, writing in it the link to the location the Client should go to, and the HTTP Status Codes associated with redirections (3xx) such as 301 Moved Permanently or 307 Temporary Redirect among others.

When the browser recieves the Location header or one of those Status Codes it knows it need to follow the Location link, so it does, and you'll find yourself in the "correct" page.
