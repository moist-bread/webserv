More about this page:

Auto Indexing is one of the options in the configuration file that allows you to make a Directory Listing page out of a specified directory.
What that becomes is a page that lists all the files inside that folder.
To create them the Server the dynamically writes them from a basic template.
The links above will take you to some examples.
